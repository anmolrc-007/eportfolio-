The Bootstrap CSS Framework

Bootstrap Framework

What is Bootstrap?

Bootstrap is the world's most popular open-source CSS framework. Created by Twitter, it provides pre-built, responsive components and a powerful grid system that lets developers build consistent, mobile-first websites rapidly — without writing all CSS from scratch.

Mobile First

Bootstrap is designed mobile-first. Styles apply to small screens by default, with breakpoints (sm, md, lg, xl, xxl) progressively enhancing for larger screens.

Grid System

A flexible 12-column grid system using containers, rows, and columns. Columns can be sized differently at each breakpoint for complex, responsive layouts.

Ready Components

Dozens of pre-built UI components: navbar, cards, modals, alerts, carousels, forms, buttons, badges, accordions, and more. Each is accessible and responsive by default.

Utility Classes

Hundreds of single-purpose utility classes for spacing (m-3, p-4), colours (text-primary), display (d-flex), and more — removing the need for custom CSS in many cases.

How to Include Bootstrap

HTML — Including Bootstrap via CDN 
<head>
  <!-- Bootstrap CSS -->
  <link rel="stylesheet"
    href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css">
</head>

<body>
  <!-- Your content using Bootstrap classes -->
  <div class="container mt-4">
    <h1 class="display-4 text-primary">Hello Bootstrap!</h1>
  </div>

  <!-- Bootstrap JS (for interactive components) -->
  <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
</body>

Bootstrap Grid System

The grid divides the page into 12 columns. You choose how many columns each element spans using classes like col-6 (half width) or col-md-4 (one-third on medium+ screens).

HTML — Bootstrap Grid Examples 
<div class="container">

  <!-- Two equal columns on all screen sizes -->
  <div class="row">
    <div class="col-6">Left Half</div>
    <div class="col-6">Right Half</div>
  </div>

  <!-- Stacks on mobile, 3 cols on medium+ screens -->
  <div class="row">
    <div class="col-12 col-md-4">Card 1</div>
    <div class="col-12 col-md-4">Card 2</div>
    <div class="col-12 col-md-4">Card 3</div>
  </div>

</div>

Bootstrap Components

Cards

Flexible content containers with headers, body, footers, images, and lists. Used for blog posts, products, user profiles. Class: card, card-body, card-title.

Navbar

Responsive navigation bar that collapses to a hamburger on mobile. Includes brand, links, dropdowns. Classes: navbar, navbar-expand-lg, navbar-toggler.

Modals

Dialog boxes that overlay the page. Use for confirmations, login forms, image previews. Triggered by data attributes: data-bs-toggle="modal". No custom JS needed.

Alerts

Feedback messages for the user. Colour-coded by context: alert-success, alert-danger, alert-warning, alert-info. Can be dismissible.

HTML — Bootstrap Card Component 
<div class="card" style="width: 18rem;">
  <img src="image.jpg" class="card-img-top" alt="...">
  <div class="card-body">
    <h5 class="card-title">Card Title</h5>
    <p class="card-text">Some description text here.</p>
    <a href="#" class="btn btn-primary">Go somewhere</a>
  </div>
</div>
