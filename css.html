Cascading Style Sheets

CSS — The Style

What is CSS?

CSS (Cascading Style Sheets) controls the visual presentation of HTML. It defines colours, fonts, spacing, layout, animations, and responsiveness. The word cascading refers to how styles are prioritised and inherited from parent to child elements.

Colors & Typography

CSS supports hex (#7c5cfc), RGB (rgb(124,92,252)), and named colours. Typography properties control font-family, font-size, font-weight, line-height, and letter-spacing.

Layout Systems

Modern CSS layout is dominated by Flexbox (one-dimensional) and Grid (two-dimensional). These replaced older float-based layouts and make responsive design far simpler.

Responsive Design

Media queries allow different styles for different screen sizes: @media (max-width: 768px) { ... }. This is the foundation of mobile-first design.

Animations & Transitions

CSS transition smoothly animates property changes. @keyframes defines complex multi-step animations — all without JavaScript. Keeps interactivity performant and smooth.

Three Ways to Add CSS

CSS — Syntax & Application Methods 
/* 1. EXTERNAL — Best practice. Separate .css file linked in <head> */
p {
  color: #333;
  font-size: 16px;
}

/* 2. INTERNAL — <style> tag inside <head> */
h1 { color: navy; }

/* 3. INLINE — style attribute directly on element */
<p style="color: red; font-weight: bold;">Text</p>

/* CSS Variables — reusable custom properties */
:root {
  --primary-color: #7c5cfc;
  --font-size-lg: 1.25rem;
}
button { background: var(--primary-color); }

CSS Selectors

Selectors determine which HTML elements a CSS rule applies to. Understanding selector specificity is key to avoiding style conflicts.
CSS — Selector Types 
/* Element selector — targets all <p> tags */
p { color: gray; }

/* Class selector — targets class="highlight" */
.highlight { background: yellow; }

/* ID selector — targets id="main-title" (unique) */
#main-title { font-size: 2rem; }

/* Descendant — p inside a .card */
.card p { padding: 1rem; }

/* Pseudo-class — button when hovered */
button:hover { background: blue; }

/* Pseudo-element — first line of paragraph */
p::first-line { font-weight: bold; }

/* Specificity order: ID (1,0,0) > Class (0,1,0) > Element (0,0,1) */

The CSS Box Model

Every HTML element is a rectangular box. The box model defines how space is calculated: content → padding → border → margin.
  Visual Box Model

CSS — Box Model Properties 
.box {
  width: 300px;         /* Content width */
  padding: 20px;        /* Space inside the border */
  border: 2px solid #333; /* Visible edge */
  margin: 16px auto;   /* Space outside (auto = centred) */
  box-sizing: border-box; /* Include padding in width calc */
}

CSS Flexbox

Flexbox makes it easy to align and distribute elements along a single axis (row or column). It's the most commonly used layout method in modern web development.
CSS — Flexbox Layout 
.container {
  display: flex;              /* Activate Flexbox */
  flex-direction: row;        /* row | column */
  justify-content: center;   /* Main axis alignment */
  align-items: center;       /* Cross axis alignment */
  gap: 1rem;                  /* Space between items */
  flex-wrap: wrap;            /* Wrap to next line if needed */
}
.item {
  flex: 1;                    /* Grow to fill space equally */
}
